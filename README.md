# Site ABTT Equipe — Netlify

Projeto estático pronto para publicar no Netlify.

## Arquivos principais
- `index.html`: página pública
- `admin.html`: painel administrativo
- `styles.css`: visual premium
- `script.js`: menu, tradução, conteúdo dinâmico e painel
- `assets/abtt-logo-premium.jpg`: logo atualizada

O painel permite cadastrar atletas, notícias, eventos, imagens e vídeos. Nesta versão estática para Netlify, os conteúdos são salvos no navegador via `localStorage`. Para um painel com login real e dados visíveis para todos os visitantes, será necessário integrar backend, banco de dados ou CMS como Decap CMS/Supabase/Firebase.

## Publicação no Netlify
Envie a pasta do projeto ou arraste este ZIP no painel do Netlify.
